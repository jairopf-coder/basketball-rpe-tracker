# 🏀 Basketball RPE Tracker

Aplicación web colaborativa para el seguimiento del RPE (Rate of Perceived Exertion) en equipos de baloncesto.

## 🌟 Características

- ✅ **Sincronización en tiempo real** entre múltiples usuarios
- 📊 **Dashboard con analíticas avanzadas**
- 👥 **Gestión de jugadores** con categorías de edad
- 📅 **Calendario de sesiones** con vista mensual
- 🏥 **Seguimiento de lesiones** y cargas
- 🔮 **Predicción de riesgos** basada en tendencias
- 📱 **PWA** - Instalable en móviles y tablets
- 💾 **Backup/Restore** de datos
- 📥 **Exportación a CSV/JSON**

## ⚡ Inicio Rápido

### Desarrollo Local

```bash
# Clonar repositorio
git clone https://github.com/TU_USUARIO/basketball-rpe-tracker.git
cd basketball-rpe-tracker

# Configurar Firebase
# 1. Crear proyecto en https://firebase.google.com
# 2. Editar firebase-config.js con tus credenciales

# Iniciar servidor local
node server.js

# Abrir en navegador
# http://localhost:3000
```

### Publicar en GitHub Pages

1. Sigue el [CHECKLIST.md](CHECKLIST.md)
2. Tu app estará en: `https://TU_USUARIO.github.io/basketball-rpe-tracker/`

## 🛠️ Tecnologías

- **Frontend**: Vanilla JavaScript (sin frameworks)
- **Backend**: Firebase Realtime Database
- **Hosting**: GitHub Pages (gratuito)
- **PWA**: Service Worker para funcionalidad offline
